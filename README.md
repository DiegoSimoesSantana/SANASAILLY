# Sana Saily Portal

Portal web em React para divulgar a cantora e compositora Sana Saily com foco em eventos, músicas, composições, fotos e contato comercial.

## Como rodar

1. Instale as dependências.
2. Rode `npm run dev`.
3. Abra a URL exibida pelo Vite.

## Assets locais

O projeto procura estes arquivos no diretório `public/`:

- `sana_saily_logo_instagram_post.png`
- `sana_saily_logo_1500x1500_transparent.png`

Se ainda não estiverem disponíveis, o site mostra conteúdo de fallback para manter a experiência funcional.

## Scripts

- `npm run dev` - ambiente de desenvolvimento
- `npm run build` - build de produção
- `npm run preview` - pré-visualização do build
